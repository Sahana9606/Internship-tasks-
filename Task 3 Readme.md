Overview

This task focuses on using SQL to extract, manipulate, and analyze structured data from a relational database.
You will practice core SQL concepts such as filtering, joining, aggregations, subqueries, creating views, and optimizing performance using indexes.

The goal is to build confidence working with SQL for real-world data analysis scenarios.

Objective

Use SQL queries to extract, analyze, and summarize data from a database.

Tools Used

You may use MySQL, PostgreSQL, or SQLite.
Any database client (MySQL Workbench, pgAdmin, DBeaver, SQLiteStudio) can be used to run the queries.
